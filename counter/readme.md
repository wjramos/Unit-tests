# Counter

## Example markup:

```
<span
    class="js-counter"
    data-count-endpoint = "https://www.rei.com/endpoint.json"
    data-count-property = "total"
    ( data-count-speed = "20.5s" )
    ( data-count-update = "5 min" )
    >

    [ Initial number ]

</span>
```

`( )` - Optional parameters


- Available sitewide - initiated with `.js-counter` class
